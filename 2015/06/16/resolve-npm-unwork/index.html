解决国内使用NPM/GEM下载速度缓慢问题

  • nodejs
  • npm

大天朝访问国外的网站隔三差五就会被墙，国内很多开发者在下载一些开发包的时候常常由于天朝的高墙下载速度缓慢乃至不能下载。

在此推荐一个NPM源切换插件NRM

安装

npm install -g nrm

  • nrm 是一个 NPM 源管理器，允许你快速地在如下 NPM 源间切换：
    • npm
    • cnpm
    • strongloop
    • european
    • australia
      • nodejitsu
    • taobao

      使用

你可以通过执行 nrm test来测试对你延迟最小的源,国内速度最佳的应该就是taobao的源了，使用时只需执行nrm use taobao即可.

更详细资料可查看：http://segmentfault.com/a/1190000000473869
